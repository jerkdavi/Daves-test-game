Are you ready to start the game?

Click on the screen to start the game!

Use arrow keys or buttons to move on the map.

Every room has a key of its own name inside of it.

Goal of the game is to reverse the keys in opposition to the room names.
The one key that can't be opposed must be in your posesion.

Type help for help!

Here is a list of commands you can try:

help - returns all available commands
goal - returns goal of the game
diary - returns all commands you tried
clrdiary - clears all entries from diary
reset - restarts the game
look - looks around the room
take object name - takes object from room
take all - take all objects from room
drop object name - drops object in room
drop all - drops all objects in room
inventory - displays contents of your inventory

north - tries to go north
south - tries to go south
east - tries to go east
west - tries to go west

Congratulation on completing the game!
